What is this?

Planet Pulse is a service created to make NOAA climate data widely available. The Planet Pulse API provides an endpoint to query the latest NOAA atmospheric measurements.

Why not get the data directly from NOAA?

Certain atmospheric measurements made by NOAA are only available through their AFTP servers as text files. Therefore, if you wanted to know the atmospheric CO2 concentration this past month, you would have to download a text file of all monthly CO2 measurements taken since 1974 and parse it yourself. Instead, Planet Pulse does this for you and stores the data in our database. When you want to retrieve data, you can make a call to our API and download only the data you need.

Where can I learn more?

Technical documentation and the source code for this project can be found on the Planet Pulse GitHub.

All data is provided by NOAA.
